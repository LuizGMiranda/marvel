# Marvel
Painel de busca de personagens consultando a Marvel API

## Tecnologias
- React
- Redux
- Axios
- React-hot-toast
- React-loading-skeleton
- TailwindCSS

## Comandos
Instalar pacotes
```cmd
npm install
```

Rodar o projeto
```cmd
npm run start
```

Rodar testes
```cmd
npm run test
```
